# Oblívio RPG - Toolbox

Um ecossistema de ferramentas digitais desenvolvido para mestres e jogadores do sistema **Oblívio RPG**. 

O projeto combina um terminal de dados imersivo com um gerador procedural de masmorras (Buracos).

## 1. Módulos do Projeto
### 🃏 Gerador de Buracos (Oblívio)
Gerador procedural de mapas baseado nas regras oficiais de exploração.
- **Geração em Lote:** Cria masmorras de 10, 20 ou 30 cartas instantaneamente.
- **Algoritmo Procedural:** Expansão orgânica em eixos X e Y (Norte, Sul, Leste, Oeste).
- **Interatividade:** Sistema Drag-and-Drop para reposicionamento e clique para leitura de descrição.

## 🚀 Como Rodar

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado.
